Class CFCoreClient

The CFCoreClient class provides a client for the CurseForge Core API (v1).

Request methods return a Promise that resolves to ResponseData.

Methods may throw a CFCoreInternalServerError if the server returns an error.

getVersionTypes

  • getVersionTypes(gameId: number): Result<GetVersionTypesResponse, never>

  • Get Version Types

    GET /v1/games/{gameId}/version-types

    Get all available version types of the specified game.

    A private game is only accessible to its respective API key.

    Currently, when creating games via the CurseForge Core Console, you are limited to a single game version type. This means that this endpoint is probably not useful in most cases and is relevant mostly when handling existing games that have multiple game versions such as World of Warcraft and Minecraft (e.g. 517 for wow_retail).

    Returns

    The version type data.

    Throws

    CFCoreNotFoundError: If the game is private or the game does not exist.

    Parameters

      A game unique id
